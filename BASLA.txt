🎯 EXTENSİON'I TARAYICINIZA YÜKLEYİN:

┌─────────────────────────────────────────────────────────────┐
│  CHROME İÇİN                                                │
├─────────────────────────────────────────────────────────────┤
│  1. Chrome'u açın                                           │
│  2. Adres çubuğuna yazın: chrome://extensions               │
│  3. Sağ üstte "Developer mode" açın                         │
│  4. "Load unpacked" tıklayın                                │
│  5. "dist-chrome" klasörünü seçin                           │
│  6. ✅ Tamamlandı! Yeni sekme açın.                         │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  FIREFOX İÇİN                                               │
├─────────────────────────────────────────────────────────────┤
│  1. Firefox'u açın                                          │
│  2. Adres çubuğuna yazın: about:debugging                   │
│  3. "This Firefox" sekmesine gidin                          │
│  4. "Load Temporary Add-on..." tıklayın                     │
│  5. "dist-firefox" klasöründeki manifest.json seçin         │
│  6. ✅ Tamamlandı! Yeni sekme açın.                         │
│                                                             │
│  ⚠️  NOT: Firefox'ta geçici eklenti her açılışta           │
│      tekrar yüklenmeli.                                     │
└─────────────────────────────────────────────────────────────┘
