## Installation

[(YT VIDEO) How to Install and Use New Tab Desktop UI?](https://www.youtube.com/watch?v=I-3GmfDBz4M)

### Clone the repo

```sh
git clone https://github.com/cenkaymartin/new-tab-desktop-ui.git
```

### Install packages

```sh
npm install
```

## Usage

### Start dev server

```sh
npm run dev
```

### Run tests once

```sh
npm run test
```

### Run tests and watch for changes

```sh
npm run test:watch
```

## Type Checking

### Check types with TypeScript

```sh
npm run tsc
```

## Linting

### Lint with ESLint

```sh
npm run lint
```

### Format with Prettier

```sh
npm run format
```

## Building

### Build web demo

```sh
npm run build
```

### Preview web build

```sh
npm run preview
```

### Build Chrome

```sh
$env:PROJECT="chrome"; npm run build
```

### Build Firefox

```sh
$env:PROJECT="firefox"; npm run build
```
